Flashing Core Module 2
The new Core Module 2 is not using DFU mode anymore. We added new flashing over FTDI chip and virtual serial port over USB. This means that the flashing procedure is now the same as with the USB Dongle. Please, use the --device <PORT> (e.g. COM4) parameters instead of the former --dfu parameter.

This change improved firmware upload speed and no need to press any button.

Also the UART2 output is connected to this virtual serial port, so you can use bc_log_* functions to debug your code.
